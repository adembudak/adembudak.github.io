Regular Types

20/3/22

A type is regular if it has following properties:

1- default constructible
2- copyable/movable
3- destructible
4- equality comparable

std::string is a regular type:

std::thread is not because it’s designed to be not copyable.

Regular types and procedures are one of core concept of the STL. Most of the types are regular in STL, and designing yours so will make your type composable with them.
